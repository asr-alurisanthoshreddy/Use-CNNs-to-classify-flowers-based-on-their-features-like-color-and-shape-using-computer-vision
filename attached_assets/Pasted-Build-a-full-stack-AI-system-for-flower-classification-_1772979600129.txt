Build a full-stack AI system for flower classification and botanical knowledge inference using the Hybrid Spectral-Spatial Attention Network (HSSAN) architecture.

The system must classify flowers and also infer phytochemical properties and geographic distribution.

Use Python + PyTorch + Hugging Face APIs.
If required knowledge or datasets are not available through Hugging Face, use Google Gemini API as a fallback.

Core System Features

The system must:

Classify flower species from images.

Simulate spectral features from RGB images.

Use spectral-spatial CNN architecture.

Apply channel attention (Squeeze-Excitation).

Predict phytochemical compounds.

Predict geographic distribution.

Provide explainable AI attention maps.
